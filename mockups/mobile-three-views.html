How This Works:

  • Single content area: All three views (Lists, Master, Meals) display in the same main content space
  • Toggle switches views: Clicking Lists/Master/Meals in the header switches the content below, NOT opening a modal
  • Only one active at a time: Only one toggle button is highlighted at any moment
  • Lists view: Swipeable individual shopping lists with pagination dots
  • Master view: Aggregated view of all unchecked items grouped by list
  • Meals view: Inline meal planner showing calendar with planned meals (can scroll through days)
  • Mobile only: This behavior is for mobile. Desktop keeps the floating "Meals" button that opens a modal
  • State preservation: When switching between views, each view remembers its state (e.g., which list you were viewing in Lists mode)
